## Key Implementation Notes:

### Fixed TypeScript Issues:
- CSS imports: Added css.d.ts for module declarations
- Button/Badge: Replaced class-variance-authority with simple object-based variants
- InputBar: Changed input to textarea for rows attribute
- Message types: Created proper Message interface with role, content, isLoading
- RightPanel: Changed from Sidebar duplicate to proper panel component
- Sidebar: Fixed activeGoal computation from goals array
- Providers: Commented out react-query-devtools (not installed)

### State Management:
- React Query: For server state (chat API calls)
- Zustand: For client state (session plan, goals, experiments, profile)

### Design System:
- Art Deco theme with custom colors: deco-gold, deco-bronze, deco-navy, deco-cream, deco-teal
- No external component libraries (custom Button, Badge, Card, etc.)
- Tailwind CSS with custom configuration

### Component Props:
- Sidebar: { isOpen: boolean, onClose: () => void }
- MessageBubble: { message: Message }
- RightPanel: No props (self-contained)
- All UI components use forwardRef pattern
